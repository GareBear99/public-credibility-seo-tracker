## Suggested PR Lines

### awesome-juce / sites.txt style

```text
https://github.com/GareBear99/FreeEQ8 - Free open-source JUCE/C++ EQ plugin for VST3/AU workflows with EQ8-style parametric mixing, spectrum analysis, and advanced DSP direction.
https://github.com/GareBear99/FreeVox8 - Free open-source JUCE/C++ vocoder and spectral voice-processing plugin for creative vocal effects and experimental sound design.
```

### OpenAudio style

```text
FreeEQ8 | Effect | JUCE/C++ | VST3/AU | Free open-source EQ8-style parametric EQ plugin for producers and developers.
FreeVox8 | Effect | JUCE/C++ | VST3/AU | Free open-source vocoder and spectral voice-processing plugin for creative vocal sound design.
```

## PR Checklist

Before opening any PR:

- README has install/build instructions.
- README has screenshots or demo images.
- Repo has LICENSE clearly visible.
- Latest release/build status is clear.
- Plugin formats are stated honestly.
- “Free/open-source” wording is consistent.
- No impossible claims; advanced features should be described as current, planned, or directional.

## Internal Rule

Do not submit to awesome lists until LibHunt core graph is done and GitHub topics are cleaned. Awesome-list maintainers are gatekeepers; the repo needs to look polished before requesting inclusion.
