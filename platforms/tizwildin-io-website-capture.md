## What this page is

The `.io` ecosystem page is the public front door for the TizWildin / GareBear99 audio ecosystem. It is not only a sample-pack router. It is the public plugin, dashboard, list, pack, visualizer, release, and creator-resource hub.

The page/repo presents the HUB as:

- a VST3/AU plugin and standalone app
- a web dashboard
- a plugin/update/version controller
- a one-click install / install-all route
- a free-version-first delivery surface
- the public discovery point for plugins, lists, sample packs, visualizers, release vaults, and creator surfaces
